# Visualisasi Data: Analisis Komparatif Sentimen Pengguna Sistem Manajemen Proyek

---

## 1. Tujuan Analisis

Dokumen ini memuat visualisasi data terkait analisis sentimen pengguna terhadap tiga platform Sistem Manajemen Proyek utama: **Trello**, **Jira**, dan **ClickUp**. Tujuan dari visualisasi ini adalah untuk memetakan tingkat kepuasan pelanggan secara proporsional dan mengidentifikasi platform dengan penerimaan pasar terbaik berdasarkan data ulasan B2B.

---

## 2. Sumber Data dan Metodologi

Data yang digunakan dalam analisis ini diekstrak dari platform agregator ulasan perangkat lunak B2B ([G2.com](http://G2.com)) pada periode **April 2026**.

Total sampel data mencakup **33.267 ulasan riil**, dengan rincian:

| Platform | Jumlah Ulasan | Skor Agregat |
|----------|--------------|--------------|
| Trello   | 13.985       | 4.4 / 5.0    |
| ClickUp  | 11.626       | 4.7 / 5.0    |
| Jira     | 7.656        | 4.3 / 5.0    |

Metodologi visualisasi menggunakan pendekatan **100% Stacked Bar Chart** (Diagram Batang Bertumpuk 100%). Pendekatan ini dipilih untuk menormalisasi perbedaan jumlah ulasan absolut antar platform, sehingga perbandingan rasio sentimen (Skala Bintang 1 hingga 5) dapat dievaluasi secara adil dan objektif.
